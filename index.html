Projects

Rate my Course December 2023

Rate my course is a web application developed to assist students in finding the best course for their schedule. Users can rate courses and view rating from other students. In terms of development I was tasked with the front end development of all aspects of the user profile. I completed this task by using React and TailwindCSS.

PlayerLink May 2023

PlayerLink is a web application that was created using blazor following the steps of task centered system design. The goal of the project was to design an application that would allow gamers with similar personalities and goals to connect with each other. This project was created using HTML, and CSS for the front end and C# for the back end.

Fitness Tracker November 2022

This project was created in order to develop a greater understanding of SQL databases. The fitness tracker is a web based application that allows users to keep track of what they eat and what exercise they complete however, it also provides users with the ability to connect with a professional whether it be a nutritionist or a personal trainer. This project was developed using HTML, CSS, PHP, Javascript and, mySQL for the DBMS

Sorry Board Game August 2021

A recreation of the popular board game "Sorry" that was created in order to develop skills in object-oriented programming. The game implements the same rule set as "Sorry" and also adds the ability to play against up to 4 players or an AI player. This project was developed in Java using the javafx library for the front end.
